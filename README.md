# Analytics Dashboard

A dark-themed analytics dashboard built with React, TypeScript, and Recharts.

## Live Demo
Add Vercel link here after deployment

## Tech Stack
- React + TypeScript
- Vite
- TailwindCSS v3
- Recharts
- Axios

## Features
- Summary metric cards
- Line and bar charts
- Filterable data table
- Fully responsive dark UI

## Run Locally
```bash
npm install
npm run dev
```
